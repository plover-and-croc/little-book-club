# **LITTLE BOOK CLUB \- WEBSITE STRUCTURE, LAYOUT & ELEMENT STYLE SPECIFICATION**

## **Design Objective**

Create a clean, friendly, trustworthy website for Little Book Club.

The site should feel like a warm family reading brand and home library builder, not a discount retailer, marketplace, or second-hand book warehouse.

The design should be simple, modern, easy to navigate, and conversion-focused.

---

# **OVERALL DESIGN DIRECTION**

## **Look and Feel**

The website should feel:

* Clean  
* Warm  
* Friendly  
* Family-focused  
* Trustworthy  
* Light and spacious  
* Educational without feeling academic  
* Premium but accessible  
* Playful in small details, not childish overall

Avoid:

* Cluttered ecommerce layouts  
* Heavy discount styling  
* Marketplace-style grids  
* Bright aggressive sales banners  
* Too many competing CTAs  
* Overly corporate design  
* Overly childish design

---

# **PAGE WIDTH AND SPACING**

## **Main Container**

Use a centered max-width container for most content.

Recommended:

* Max width: 1140px to 1200px  
* Horizontal padding: 24px desktop, 16px mobile

## **Section Spacing**

Use generous vertical spacing between sections.

Recommended:

* Desktop section padding: 80px to 100px  
* Mobile section padding: 48px to 64px

The site should feel calm and easy to read.

---

# **TYPOGRAPHY STRUCTURE**

Brand font choices will be defined separately.

## **Heading Style**

Headings should be clear, warm, and confident.

Use:

* Large hero headline  
* Clear section headings  
* Short subheadings  
* Avoid long headline blocks

## **Body Text**

Body copy should be easy to scan.

Use:

* Short paragraphs  
* Comfortable line height  
* Maximum readable text width  
* Friendly but polished language

Avoid dense text blocks.

---

# **BUTTONS AND CTAS**

## **Primary CTA**

Text:

Build Your Bundle

Use for the main conversion action.

Style:

* Visually prominent  
* Rounded corners  
* Solid filled button  
* Warm, friendly colour from brand palette  
* Clear hover state  
* Large enough for mobile tapping

## **Secondary CTA**

Text:

Join the Club

Use for mailing list/community action.

Style:

* Outline button or softer filled button  
* Less visually dominant than primary CTA  
* Should scroll to the Join the Club section on homepage

## **CTA Rules**

Each section should usually have one clear action.

Avoid placing too many buttons close together.

Primary action should always be obvious.

---

# **ORDER FLOW LAYOUT STYLE**

The Build Your Bundle page should use a two-column layout.

Desktop:

* Left 66%: bundle builder form  
* Right 33%: sticky order summary

Mobile:

* Form first  
* Order summary collapses or appears below selections  
* Checkout button should remain easy to access

Builder elements should use:

* Card-based selections  
* Pill buttons for interests  
* Clear selected states  
* Friendly microcopy  
* Progress indicators if needed

Avoid making it feel like a long form.

It should feel like creating a personalised reading adventure.

* Benefits  
* Steps  
* Bundles  
* Interest selections  
* Account dashboard blocks

Recommended style:

* Rounded corners  
* Soft shadow or light border  
* Generous padding  
* Clear hover state

* Under hero

Use accessible design basics:

* Readable font sizes  
* Keyboard-friendly navigation  
* Proper button states

* Clear form labels  
* Semantic headings

# **BRAND EXPERIENCE RULE**

Every design decision should support this feeling:

Little Book Club helps families raise readers and build home libraries children love.

The website should never feel like:

* A discount book shop  
* A used-book marketplace  
* A clearance sale  
* A generic ecommerce template

It should feel like:

* A trusted family reading brand  
* A reading club  
* A curated home library service  
* A warm community for families
